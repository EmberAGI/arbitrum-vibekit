## Key Takeaways

### âœ… append=true behavior:

- **NEVER deletes any artifacts**
- All artifacts (same type OR different types) accumulate
- Perfect for comparison views, lists, galleries

### âŒ append=false behavior:

- **ONLY deletes artifacts with MATCHING toolName**
- Different types are unaffected
- Perfect for updating quotes, status, recommendations

- Pendle artifacts only replace other pendle artifacts
